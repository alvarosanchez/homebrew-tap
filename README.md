# Alvarosanchez Tap

## How do I install these formulae?

`brew install alvarosanchez/tap/<formula>`

Or `brew tap alvarosanchez/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "alvarosanchez/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
